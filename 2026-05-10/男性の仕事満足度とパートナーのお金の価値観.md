# 男性の仕事満足度とパートナーのお金の価値観

男性の仕事の満足度はパートナーとのお金の価値観に左右されるのではないか、という研究の話が面白かったのでまとめてみます。仕事の満足度は本人の給料や出世欲だけで決まるわけではなく、パートナーとお金をどう意味付けているかが噛み合っているかにも左右される、といった話ですね。

この研究はアメリカとイギリスに住む共働きカップルを対象にしたもので、調査の手順は以下の通りです：

1. まず全員に「金は成功の証だと思うか」を質問する
2. その10日から14日後に「今の仕事は自分が求めているものを満たしているか」を調べる
3. 両者の回答から、どのような傾向が見られるかをチェックする

その結果がどうだったかというと、男性の場合、自分と女性パートナーの「お金＝成功」という価値観が一致しているほど、仕事への満足度が高かったそうです。もう少し詳しく言うと、夫婦がどちらも「お金こそ成功の証だ」という思考を持っている場合は男性の仕事の満足度は高く、逆もしかりで、夫婦がどちらも「お金は成功とは関係ないよね」と思っている場合も男性の仕事の満足度は高いという傾向があります。

つまり、夫婦のお金の価値観が合っているほど男性の仕事の満足度が高くなる、という研究結果があるようです。一方で、最も仕事の満足度が低いのは、カップルの価値観が中途半端にずれているケースだったそうです。

研究チームは「完全に対立しているわけではないけれど、なんとなく噛み合っていない状態」が最も良くないのかもしれないと指摘しています。

自分とパートナーは違うタイプだと認識はできるものの、中途半端にずれていると、以下のような曖昧な不安が残ってしまいます。

1. 自分はもっと稼ぐべきなのだろうか？
2. パートナーは本当は収入を求めているのだろうか？

こうした不安が、日々の仕事の満足度をがっつり下げてしまうということがあるようです。ちなみに女性にはこのような関係が見られなくて、これも面白いところだなと思いました。

研究チームは、これは男女に関わる社会的な期待の違いかもしれないと話しています。男性は稼ぎ手であるという、お金に対するプレッシャーが強い一方で、女性の場合は仕事の満足度を考える際に、収入だけではなく以下のような要素も関わってくるからではないかとのことでした。

1. 柔軟性
2. 家庭との両立
3. ケア責任
4. 職場環境
5. 専業主婦として頑張るという選択肢

確かにこれらはありそうだな、と感じました。特に男性は「もっと稼がなければならない」という社会的なプレッシャーを勝手に背負いやすいから、自分にとって稼ぐことは何かを考えておくのは大事なことだろうな。

## 穴埋め問題

### 問題

1. 男性の仕事満足度は、本人の給料や出世欲だけでなく、パートナーとお金をどう（1）しているかにも左右される。
2. 研究では、アメリカとイギリスに住む（2）カップルが対象になった。
3. 最初の調査では、「金は（3）の証だと思うか」を質問した。
4. 男性の場合、自分と女性パートナーの「お金＝成功」という価値観が（4）しているほど、仕事満足度が高かった。
5. 夫婦がどちらも「お金は成功とは関係ない」と思っている場合も、男性の仕事満足度は（5）傾向があった。
6. 最も仕事満足度が低いのは、カップルの価値観が中途半端に（6）ているケースだった。
7. 中途半端なズレは、「自分はもっと稼ぐべきなのだろうか」という（7）な不安につながる。
8. 女性には同じ関係が見られにくく、その理由として男女に関わる（8）の違いが考えられている。
9. 男性は「稼ぎ手である」という、お金に対する（9）を背負いやすい。
10. 女性の仕事満足度には、収入だけでなく柔軟性・家庭との両立・（10）責任なども関わると考えられている。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 意味付け |
| （2） | 共働き |
| （3） | 成功 |
| （4） | 一致 |
| （5） | 高い |
| （6） | ずれ |
| （7） | 曖昧 |
| （8） | 社会的な期待 |
| （9） | プレッシャー |
| （10） | ケア |

### 心理学ポイント

- 仕事満足度は、個人の能力や収入だけでなく、パートナーとの価値観の一致にも影響される。
- 「完全な違い」よりも「なんとなく噛み合わないズレ」が不安を生みやすい点が重要である。
