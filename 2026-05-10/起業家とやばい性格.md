# 起業家とやばい性格

起業家には「やばい性格」の人が多いという話が面白かったので、その内容をまとめます。

起業家にはナルシシズムやサイコパシーの傾向を持つ人が多いのではないかと指摘されており、研究の結果、起業家精神が旺盛だったのは以下のような属性の人たちでした：

1. 若い男性
2. 自営業者
3. 好奇心が強い人
4. 真面目な人
5. 外向的な人
6. ナルシシズムが高い人
7. サイコパシー傾向が高い人

起業というのは、自分のアイデアには価値があるとか、自分ならできるといった「万能感」が必要ですから、意外とナルシシズムが役立つ場面は多いのでしょうね。

また、サイコパス気質が強い人はリスクを恐れないため、起業に伴う恐怖を乗り越えるのも容易です。だからこそ、サイコパシー傾向もまた、起業においてはアクセルとして働くケースがあるのだと思います。

この研究からわかるのは、起業家精神というのは「良い性格」と「悪い性格」のミックスだということです。確かにポジティブな特性が必要なのですが、同時にダークな特性も入り混じった、かなり複雑な心理パッケージなのではないかという話ですね。

世の中を変える人が、必ずしも性格が良い人とは限りません。むしろ「悪人こそが成功するのではないか」と個人的に思ってしまうタイプなので、非常にリアルな結論だと感じます。

## 穴埋め問題

### 問題

1. 起業家には、ナルシシズムや（1）の傾向を持つ人が多いのではないかと指摘されている。
2. 研究で起業家精神が旺盛だった属性には、若い男性・自営業者・（2）が強い人などが含まれていた。
3. 起業には「自分のアイデアには価値がある」「自分ならできる」という（3）が必要になることがある。
4. ナルシシズムは、起業において役立つ場面が（4）と考えられている。
5. サイコパス気質が強い人は、（5）を恐れにくい。
6. サイコパシー傾向は、起業において（6）として働くケースがある。
7. 起業家精神は、「良い性格」と「悪い性格」の（7）だとまとめられている。
8. 世の中を変える人が、必ずしも（8）が良い人とは限らない。

### 答え

| 番号 | 答え |
|---|---|
| （1） | サイコパシー |
| （2） | 好奇心 |
| （3） | 万能感 |
| （4） | 多い |
| （5） | リスク |
| （6） | アクセル |
| （7） | ミックス |
| （8） | 性格 |

### 心理学ポイント

- 起業家精神は、外向性や真面目さだけでなく、ナルシシズムやリスク耐性とも関係する。
- 「良い特性だけで成功する」と単純に考えず、複数の性格特性の組み合わせとして見ることが大切である。
