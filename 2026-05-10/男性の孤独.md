# 男性の孤独

## どんな問題？
近年、特に**男性は年をとるほど友達が減りやすく、孤独になりやすい**ことが研究でわかってきている。

## なぜ男性の方が孤独になりやすい？
- 男性は「弱音を吐くのはかっこ悪い」と思いがちで、悩みを人に話さない
- 仕事中心の人間関係が多く、退職すると一気に話す相手がいなくなる
- 「友達と用もなく会う」という習慣を持ちにくい（女性は理由なくお茶できる人が多い）

## 孤独の怖さ
研究では、孤独はタバコを1日15本吸うのと同じくらい健康に悪いと言われている。
- うつ病になりやすい
- 心臓病のリスクが上がる
- 寿命が短くなる

## 対策
- **用がなくても友達に連絡する習慣**をつくる（「元気？」だけでOK）
- 趣味のコミュニティに入る（共通の話題があると関係が続きやすい）
- 弱音を吐ける相手を1人でいいので持つ
- 家族以外の人間関係を意識して育てる

## ポイント
孤独は性格の問題ではなく、社会の仕組みや習慣の問題。意識して動かないと、男性は自然に孤独に近づいてしまう。

## 穴埋め問題

### 問題

1. 近年、特に男性は年をとるほど友達が（1）やすく、孤独になりやすいことがわかってきている。
2. 男性は「弱音を吐くのは（2）」と思いやすく、悩みを人に話さないことがある。
3. 仕事中心の人間関係が多いと、（3）した後に話す相手が一気に減りやすい。
4. 孤独は、タバコを1日（4）本吸うのと同じくらい健康に悪いと言われている。
5. 孤独になると、うつ病や（5）病のリスクが上がる。
6. 対策として、用がなくても友達に（6）する習慣を作ることが大切である。
7. 趣味のコミュニティは、（7）の話題があるため関係が続きやすい。
8. 孤独は性格の問題ではなく、社会の仕組みや（8）の問題でもある。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 減り |
| （2） | かっこ悪い |
| （3） | 退職 |
| （4） | 15 |
| （5） | 心臓 |
| （6） | 連絡 |
| （7） | 共通 |
| （8） | 習慣 |

### 心理学ポイント

- 孤独は「本人の性格」だけでなく、仕事中心の人間関係や弱音を吐きにくい文化とも関係する。
- 家族以外の人間関係を意識して育てることが、孤独の予防になる。
