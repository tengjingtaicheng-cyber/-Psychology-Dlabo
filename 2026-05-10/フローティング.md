# フローティング

## どんな状態？
何かに集中していて、**時間を忘れて夢中になっている状態**のこと。心理学では「フロー状態」とも呼ばれる。

## 身近な例
- ゲームに熱中していたら、気づいたら3時間たっていた
- 絵を描いていて、ご飯を食べるのを忘れた
- スポーツの試合中、まわりの音が聞こえなくなって体が勝手に動いた

## フロー状態に入る条件
1. **目標がはっきりしている**（何をすればいいかわかっている）
2. **すぐにフィードバックがある**（うまくいったか、失敗したかすぐわかる）
3. **難しすぎず簡単すぎない**（自分の実力よりちょっとだけ難しいくらい）

## なぜ大事？
フロー状態のとき、人は一番幸せを感じやすいと言われている。勉強でも仕事でも、フローに入れると成果が出やすく、しかも楽しい。

## 入りやすくするコツ
- スマホを別の部屋に置く（通知で集中が切れない）
- やることを1つにしぼる
- 「ちょっと難しいけどできそう」なレベルの課題を選ぶ

## 穴埋め問題

### 問題

1. フローティングとは、何かに集中していて（1）を忘れて夢中になっている状態である。
2. 心理学では、フローティングは（2）状態とも呼ばれる。
3. フロー状態に入る条件の一つは、（3）がはっきりしていることである。
4. フロー状態に入るには、うまくいったか失敗したかがすぐわかる（4）があることも大切である。
5. 課題の難しさは、難しすぎず簡単すぎず、自分の実力より（5）難しいくらいがよい。
6. フロー状態のとき、人は（6）を感じやすいと言われている。
7. フローに入りやすくするには、通知で集中が切れないように（7）を別の部屋に置くとよい。
8. やることを（8）にしぼることも、フローに入りやすくするコツである。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 時間 |
| （2） | フロー |
| （3） | 目標 |
| （4） | フィードバック |
| （5） | ちょっとだけ |
| （6） | 幸せ |
| （7） | スマホ |
| （8） | 1つ |

### 心理学ポイント

- フローは「目標」「すぐわかる結果」「少し難しい課題」がそろうと起こりやすい。
- 勉強でも仕事でも、集中できる環境を作ることがフローの入り口になる。
