# 外見の自己評価と職場での発言しやすさ

自分は見た目がいいと思っているほど、職場で意見を言いやすいという話があって面白いので、内容をまとめてみました。

研究チームが注目したのは、外見は社会で成功するための道具であるという信念である。簡単に言うと、見た目がいい人は周囲から好意的に扱われる、発言を聞いてもらいやすい、能力が高そうに見られるなど、外見が人間関係や仕事で有利に働く、と考える傾向のことを意味しています。

研究の分析結果によると、自分の外見を魅力的だと感じている人ほど、自分の発言や行動が職場に影響を与えると感じやすく、その結果として職場で意見を言いやすくなっているという事実が判明しました。

なので、この研究を自分に生かすとすれば、外見そのものよりも自分の状態を整えていこうぜ、そして、自分が発言しやすい心理状態を作ろうぜ、ということじゃないかなと思った。睡眠不足や服装がしっくりこない日、髪型や姿勢が気になる時、あるいは体調が悪い時などは、自分の外見評価が下がったり、人前で話したくないと感じたりすることがあります。

会議中に髪型や姿勢に気を取られてしまい、自分の発言の価値を低く見積もりやすくなるという側面もあるので、単に見た目を良くするというよりも、普段のコンディションを整えて、自分が発言しやすい心理状態にしておくことが大切なのではないかと思いました。

ご視聴ありがとうございました。服装や身だしなみは、他人に評価されるためというよりは、自分の品位を上げるための重宝する手段ぐらいに捉えておくといいんじゃないかなと思った。

## 穴埋め問題

### 問題

1. 研究チームが注目したのは、外見は社会で成功するための（1）であるという信念である。
2. 自分の外見を魅力的だと感じている人ほど、職場で（2）を言いやすい傾向がある。
3. 外見がよい人は、周囲から好意的に扱われたり、発言を（3）もらいやすいと考えられることがある。
4. 自分の外見評価が高いと、自分の発言や行動が職場に（4）を与えると感じやすくなる。
5. 睡眠不足や服装がしっくりこない日は、自分の（5）評価が下がることがある。
6. 会議中に髪型や姿勢に気を取られると、自分の発言の（6）を低く見積もりやすくなる。
7. 大切なのは、単に見た目をよくすることではなく、普段の（7）を整えることである。
8. 服装や身だしなみは、他人に評価されるためだけでなく、自分の（8）を上げる手段として捉えるとよい。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 道具 |
| （2） | 意見 |
| （3） | 聞いて |
| （4） | 影響 |
| （5） | 外見 |
| （6） | 価値 |
| （7） | コンディション |
| （8） | 品位 |

### 心理学ポイント

- 外見そのものより、「自分は発言してよい」と思える心理状態が大切である。
- 身だしなみは、他人の評価だけでなく、自分の自信や発言しやすさにも関係する。
