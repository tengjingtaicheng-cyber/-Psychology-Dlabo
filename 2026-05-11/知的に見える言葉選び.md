# 知的に見える言葉選び

パレオさんの記事では、「この人、感じ悪いな」と思われないための話し方について紹介されていた。

知的に見せたいときは、ただ難しい言葉を使えばいいわけではない。  
むしろ、言葉の選び方によっては、自信がなさそうに見えたり、えらそうに見えたりしてしまう。

## 1. 曖昧な言葉を減らす

「ちょっと」「ある意味」「たぶん」などの言葉を多く使うと、自信がなさそうに見えることがある。

たとえば、

「ある意味、それは大事かもしれないです」

と言うよりも、

「それは大事だと思います」

と言った方が、考えがはっきりしているように見える。

## 2. 強すぎる断定も避ける

一方で、「絶対に」「確実に」などの言葉を使いすぎるのもよくない。

これらの言葉は、自信があるように見えることもあるが、場合によっては「えらそう」「決めつけが強い」と受け取られる。

知的に見える人は、強く言い切るだけではなく、相手が受け取りやすい言い方を選んでいる。

## 3. 相手を見下すような言葉に注意する

「明らかに」「当然」などの言葉も注意が必要。

たとえば、

「そんなの当然でしょ」

と言われると、相手は「自分がバカにされた」と感じるかもしれない。

自分では普通に話しているつもりでも、相手には礼儀がないように見えることがある。

## カジュアルな言葉の方がよい場合もある

研究では、意外にも「完全にそうなのよ」のようなカジュアルな言葉がよい、という説もある。

これは、相手に対してかたすぎず、えらそうにも見えにくいからだと考えられる。

知的に見せるためには、難しい言葉を使うよりも、相手に嫌な感じを与えない言葉を選ぶことが大切。

## まとめ

知的に見える話し方のポイントは、次の3つ。

- 自信なさげな曖昧な言葉を減らす
- 強すぎる断定の言葉を使いすぎない
- 相手を見下すように聞こえる言葉を避ける

頭がよさそうに見える人は、強い言葉を使う人ではなく、相手が聞きやすい言葉を選べる人なのかもしれない。

## 穴埋め問題

### 問題

1. 知的に見せたいときは、ただ（1）言葉を使えばいいわけではない。
2. 「ちょっと」「ある意味」「たぶん」などを多く使うと、（2）がなさそうに見えることがある。
3. 「ある意味、それは大事かもしれないです」より、「それは（3）だと思います」のほうが考えがはっきり見える。
4. 「絶対に」「確実に」などの言葉を使いすぎると、えらそう・（4）が強いと受け取られることがある。
5. 知的に見える人は、強く言い切るだけではなく、相手が（5）やすい言い方を選んでいる。
6. 「明らかに」「当然」は、相手を（6）すように聞こえることがある。
7. カジュアルな言葉は、相手に対してかたすぎず、（7）にも見えにくい場合がある。
8. 知的に見える話し方のポイントは、自信なさげな曖昧な言葉を減らし、強すぎる（8）の言葉を使いすぎないことである。
9. 頭がよさそうに見える人は、強い言葉ではなく、相手が（9）言葉を選べる人である。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 難しい |
| （2） | 自信 |
| （3） | 大事 |
| （4） | 決めつけ |
| （5） | 受け取り |
| （6） | 見下 |
| （7） | えらそう |
| （8） | 断定 |
| （9） | 聞きやすい |

### コミュニケーションポイント

- 曖昧すぎる言葉は弱く見え、断定しすぎる言葉は強すぎて嫌な印象になりやすい。
- 知的さは、難語よりも「相手に配慮した言葉選び」で伝わる。
