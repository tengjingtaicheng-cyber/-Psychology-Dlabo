# 感じ悪く見られない話し方

## 元のテーマ

パレオ記事  
なんかこの人感じ悪いなと思われないための研究

## まとめ

人から「この人、感じ悪いな」と思われないためには、言葉の選び方が大事。

特に、知的に見せたいときほど、自信がなさすぎる言葉や、強すぎる断定の言葉は避けたほうがよい。

## 避けたほうがいい言葉

### 自信がなさそうに見える言葉

- まあ
- ある意味
- たぶん
- なんとなく

こうした言葉が多いと、相手から「この人は自分の意見に自信がなさそう」と見られやすい。

たとえば、

「まあ、ある意味これは大事だと思います」

と言うよりも、

「これは大事だと思います」

と言ったほうが、はっきりしていて伝わりやすい。

## 断定しすぎる言葉も注意

- 絶対に
- 確実に
- 必ず

こうした言葉は、自信があるように見える一方で、知的には見られにくいことがある。

理由は、世の中の多くのことには例外があるから。

「絶対にこうです」と言い切ると、相手から「本当にそこまで言えるの？」と思われることがある。

## 礼儀がないように見えやすい言葉

- 明らかに
- 当然
- 普通に考えて

これらの言葉は、相手の考えを下に見ているように聞こえることがある。

たとえば、

「当然、こっちのほうが正しいです」

と言われると、相手は「自分の考えは間違っていて当たり前と言われているみたい」と感じるかもしれない。

## 一番印象がよさそうな言い方

研究では、少しカジュアルで自然な言い方のほうが、相手に好印象を与えやすい可能性がある。

たとえば、

「完全にそうなのよ」

のような言い方は、かたすぎず、相手に共感している感じが出る。

## 大事なポイント

頭がよく見える話し方は、難しい言葉を使うことではない。

自信がなさすぎず、強く言い切りすぎず、相手を見下しているように聞こえない言葉を選ぶことが大切。

つまり、ちょうどいい話し方が一番印象をよくする。

## 穴埋め問題

### 問題

1. 人から感じ悪く見られないためには、言葉の（1）が大事である。
2. 「まあ」「ある意味」「たぶん」「なんとなく」が多いと、自分の意見に（2）がなさそうに見られやすい。
3. 「まあ、ある意味これは大事だと思います」より、「これは（3）だと思います」のほうが伝わりやすい。
4. 「絶対に」「確実に」「必ず」は、自信があるように見える一方で、（4）には見られにくいことがある。
5. 強すぎる断定が注意される理由は、世の中の多くのことには（5）があるからである。
6. 「明らかに」「当然」「普通に考えて」は、相手の考えを（6）見ているように聞こえることがある。
7. 研究では、少し（7）で自然な言い方のほうが好印象を与えやすい可能性がある。
8. 頭がよく見える話し方は、難しい言葉を使うことではなく、相手を（8）しているように聞こえない言葉を選ぶことである。
9. 一番印象をよくするのは、強すぎず弱すぎない（9）話し方である。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 選び方 |
| （2） | 自信 |
| （3） | 大事 |
| （4） | 知的 |
| （5） | 例外 |
| （6） | 下に |
| （7） | カジュアル |
| （8） | 見下 |
| （9） | ちょうどいい |

### コミュニケーションポイント

- 自信がなさすぎる言葉と、強すぎる断定の両方に注意する。
- 知的に見える話し方は、難しい言葉よりも、相手が受け取りやすい言葉選びで決まる。
