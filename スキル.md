## コミット＆Push ルール

「メッセージを打ってコミットしてください」と言われた場合、以下を自動で実行する：

1. `git add` で変更ファイルをステージング
2. 内容に合った日本語のコミットメッセージでコミット
3. `git push origin main` で GitHub に反映
