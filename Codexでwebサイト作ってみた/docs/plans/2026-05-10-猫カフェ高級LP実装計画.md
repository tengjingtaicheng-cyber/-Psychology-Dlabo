# 猫カフェ高級LP実装計画

## 目的

架空の高級猫カフェ「Maison Neko」のWebサイトを、生成済みUIモックを基準にReact + Tailwind CSSで実装する。

## 対象ページ

1. トップページ
2. 猫たち
3. 空間
4. メニュー
5. ご予約
6. 初めての方へ

## デザイン基準

- 白を基調に、黒・温かいグレー・シャンパンゴールドで構成する。
- 可愛い方向ではなく、ホテルラウンジのような静かな高級感を優先する。
- 余白を大きく取り、細い罫線、上品な写真、控えめな影で構成する。
- 角丸は6pxから8px程度に抑える。
- 日本語の読みやすさを優先し、見出しは明朝系、本文は上品なゴシック系にする。
- 猫の可愛さはアイコンや柄ではなく、写真素材と文章で表現する。

## 技術方針

- Vite + React + TypeScriptで新規サイトを構築する。
- Tailwind CSSを使用する。
- ページ切り替えはReact Routerを使う。
- 画像素材は`src/assets`に保存し、各ページで再利用する。
- アイコンは線画風SVGとして作成し、透過背景で使用する。
- アニメーションはCSSとIntersection Observerを中心に実装し、過度に派手にしない。

## 画像方針

- 生成済みモックをリファレンスとして、サイト内で使う写真素材を追加生成する。
- 必要な素材は以下。
  - トップヒーロー
  - 猫たちヒーロー
  - 空間ヒーロー
  - メニューヒーロー
  - 予約ヒーロー
  - 初めての方ヒーロー
  - 猫プロフィール画像6点
  - 空間ギャラリー4点
  - メニュー画像4点
- アイコンはSVGで透過背景にする。
