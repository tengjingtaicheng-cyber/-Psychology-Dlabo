# 認知オフローディング

認知オフローディングとは、脳内で処理すべき情報や思考を外部のツールや環境に肩代わりさせることです。

例えば：
1. 計算をスマホの電卓に任せる
2. 予定をカレンダーアプリに書き出す
3. 道を覚えずにGoogleマップを使用する

---

## 穴埋め問題

### 問題

1. 認知オフローディングとは、脳内で処理すべき情報や思考を（1）のツールや環境に肩代わりさせることである。
2. スマホの電卓に計算を任せることは、（2）オフローディングの例である。
3. 予定を（3）アプリに書き出すことも、認知オフローディングにあたる。
4. 道を覚えずに（4）を使うことは、記憶や判断を外部ツールに任せる例である。
5. 認知オフローディングでは、脳の負担を減らす一方で、外部の（5）に頼ることが増える。

### 答え

| 番号 | 答え |
|---|---|
| （1） | 外部 |
| （2） | 認知 |
| （3） | カレンダー |
| （4） | Googleマップ |
| （5） | ツール |

### 心理学ポイント

- 認知オフローディングは「頭の中でやることを外に出す」と考えると覚えやすい。
- 電卓・メモ・カレンダー・地図アプリは、身近な認知オフローディングの例である。
